Exercício 1. Considere que a dinâmica do sistema pode ser representada pela Função Transferência \(G(s)\) (que é a relação entre a entrada \(F(s)\) e a saída \(X(s)\)): \[G(s) = \frac{X}{F}(s) = \frac{s}{s^{2} + 6s + 5}.\] Calcule a resposta temporal \(x(t)\) para um impulso.
lightbulb

Dica: A transformada de Laplace do impulso \(\delta(t)\) é dada por \[F(s)=\mathcal{L}\left( \delta(t) \right) = 1\quad \text{e} \quad \mathcal{L}\left( e^{at} \right) = 1/(s - a).\]

Solução

Sabemos que \[G(s) = \frac{X}{F}(s) = \frac{s}{s^{2} + 6s + 5}.\] No domínio do Laplace, \(X(s) = G(s)F(s)\). Como \(F(s) = 1\) (impulso), temos \[X(s) = \frac{s}{s^{2} + 6s + 5}1.\] Das raízes do denominador de \(X(s)\), temos os pólos \(p_{1} = -1\) e \(p_{2} = -5\). Logo, aplicando o Método das Frações Parciais: \[X(s) = \frac{A}{s + 1} + \frac{B}{s + 5} = \frac{As + 5A + Bs + B}{(s + 1)(s + 5)} = \frac{s}{s^{2} + 6s + 5}.\] Resultando em um sistema de equações: \[ \begin{cases} A + B = 1 & (\text{termos que multiplicam o } s^{1})\\ 5A + B = 0 & (\text{termos que multiplicam o } s^{0}). \end{cases} \] Resolvendo este sistema de equações, encontramos \(A = - 1/4\), \(B = 5/4\). Assim, \[X(s) = \frac{A}{s + 1} + \frac{B}{s + 5} = - \frac{1/4}{s + 1} + \frac{5/4}{s + 5}.\] Aplicando a Transformada de Laplace Inversa, temos \[x(t) = - \frac{1}{4}e^{- t} + \frac{5}{4}e^{- 5t}.\] Podemos encontrar a mesma solução utilizando os seguintes comandos no Python Notebook. 

import sympy
sympy.init_printing()
s, t = sympy.symbols('s t')

def L(f):
    return sympy.laplace_transform(f, t, s, noconds=True)
def invL(F):
    return sympy.inverse_laplace_transform(F, s, t)

# entrada
F = 1
# Função Transferência
G = (s)/((s**2+6*s+5))
# Resposta temporal via Trans. Inversa de Laplace
X = G * F
invL(X)

Exercício 2. Considere que a dinâmica do sistema pode ser representada pela Função Transferência \(G(s)\) (que é a relação entre a entrada \(F(s)\) e a saída \(X(s)\)): \[G(s) = \frac{X}{F}(s) = \frac{1}{s + 1}.\] Calcule a resposta temporal \(x(t)\) para uma entrada degrau unitário.
lightbulb

Dica: A Transformada de Laplace do degrau unitário é dada por \(F(s) = 1/s\).

Solução

Sabemos que \[G(s) = \frac{X}{F}(s) = \frac{1}{s + 1}.\] No domínio do Laplace, \(X(s) = G(s)F(s)\). Como \(F(s) = 1/s\ \)(degrau unitário), temos \[X(s) = \frac{1}{s + 1}\cdot\frac{1}{s} = \frac{1}{s(s + 1)}.\] Das raízes do denominador de \(X(s)\), temos os pólos \(p_{1} = 0\) e \(p_{2} = -1\). Logo, aplicando o Método das Frações Parciais: \[X(s) = \frac{A}{s} + \frac{B}{s + 1} = \frac{As + A + Bs}{s(s + 1)} = \frac{1}{s(s + 1)}.\] Resultando em um sistema de equações: \[ \begin{cases} A + B = 0 & (\text{termos que multiplicam o } s^{1})\\ A = 1 & (\text{termos que multiplicam o } s^{0}). \end{cases} \] Resolvendo este sistema de equações, encontramos \(A = 1\), \(B = - 1\). Assim, \[X(s) = \frac{A}{s} + \frac{B}{s + 1} = \frac{1}{s} - \frac{1}{s + 1}.\] Aplicando a Transformada de Laplace Inversa: \[x(t) = 1 - e^{- t}.\] Podemos encontrar a mesma solução utilizando os seguintes comandos no Python Notebook.

import sympy
sympy.init_printing()
s, t = sympy.symbols('s t')

def L(f):
    return sympy.laplace_transform(f, t, s, noconds=True)
def invL(F):
    return sympy.inverse_laplace_transform(F, s, t)

# entrada
F = 1/s
# Função Transferência
G = (1)/((s+1))
# Resposta temporal via Trans. Inversa de Laplace
X = G * F
invL(X)

Exercício 3. Considere que a dinâmica do sistema pode ser representada pela Função Transferência \(G(s)\) (que é a relação entre a entrada \(F(s)\) e a saída \(X(s)\)): \[G(s) = \frac{X}{F}(s) = \frac{1}{s^{2} + s + 1}.\] Calcule a resposta temporal \(x(t)\) para um impulso.
lightbulb

Dica: A transformada de Laplace do impulso \(\delta(t)\) é dada por \[F(s)=\mathcal{L}\left( \delta(t) \right) = 1\quad \text{e}\quad \mathcal{L}\left(e^{at}\sin(bt) \right) = \frac{b}{(s - a)^{2} + b^{2}}.\]

Solução

Sabemos que \[G(s) = \frac{X}{F}(s) = \frac{1}{s^{2} + s + 1}.\] No domínio do Laplace, \(X(s) = G(s)F(s)\). Como \(F(s) = 1\ \)(impulso), temos \[X(s) = \frac{1}{s^{2} + s + 1}.\] Das raízes do denominador de \(X(s)\), temos os pólos \[p_{1} = -\frac{1}{2}+\frac{\sqrt{3}}{2}i,\quad p_{2} = -\frac{1}{2}-\frac{\sqrt{3}}{2}i.\] Como os polos são complexos conjugados, a melhor estratégia é verificar qual a Transformada de Laplace mais adequada para as relações. Podemos reescrever \(X(s)\) da seguinte maneira: \[X(s) = \frac{1}{s^{2} + s + 1} = \frac{1}{{(s +1/2)}^{2} + (\sqrt{3}/2)^2}.\] Observando a \(\mathcal{L}\left( e^{at}\sin(bt) \right) = b/((s - a)^{2} + b^{2})\), podemos encontrar os termos apropriados: \[X(s) = \frac{2\sqrt{3}}{3} \frac{\sqrt{3}/2}{{(s +1/2)}^{2} + (\sqrt{3}/2)^2}.\] onde \(a = - 1/2\) e \(b = \sqrt{3}/2\). Aplicando a Transformada de Laplace Inversa: \[x(t) = \frac{2\sqrt{3}}{3}e^{- t/2}\sin\Big(\frac{\sqrt{3}}{2}t\Big).\] Podemos encontrar a mesma sympy.init_printing() solução utilizando os seguintes comandos no Python Notebook.

import sympy
sympy.init_printing()
s, t = sympy.symbols('s t')

def L(f):
    return sympy.laplace_transform(f, t, s, noconds=True)
def invL(F):
    return sympy.inverse_laplace_transform(F, s, t)

# entrada
F = 1
# Função Transferência
G = (1)/((s**2+s+1))
# Resposta temporal via Trans. Inversa de Laplace
X = G * F
invL(X)

Exercício 4. A figura mostra um diagrama de blocos de um sistema em malha fechada composto por três blocos: o ganho \(K\), a função de transferência \(G(s)\) e a função de transferência \(H(s)\). A entrada desse diagrama de blocos é \(r(t)\) e a saída é \(x(t)\), que é a saída de \(G(s)\). A entrada do bloco \(K\) é o erro dado por \(E(s) = R(s) - H(s)\), a entrada do bloco \(G(s)\) é a saída do bloco \(K\), e a entrada do bloco \(H(s)\) é \(x(t)\), a saída do bloco \(G(s).\)

Considerando a figura abaixo, indique a faixa de ganhos \(K\) que tornam o sistema estável. Dados: \[K(s) = \frac{K}{s},\quad G(s) = \frac{s + 6}{s^{2} + 3s + 2},\quad H(s) = 1.\]

Solução

Usando digrama de blocos, encontramos a FT (slides 4-14 da Videoaula 6 e p. 14-24 de Ogata) de malha fechada:

\[\frac{X}{R}(s) = \frac{K(s)G(s)}{1 + K(s)G(s)H(s)} = \frac{\displaystyle\frac{K(s + 6)}{s(s^{2} + 3s + 2)}}{1 + \displaystyle\frac{Ks + 6K}{s(s^{2} + 3s + 2)}} = \frac{K(s + 6)}{{s^{3} + 3s}^{2} + (2 + K)s + 6K}.\]

Aplicando o critério de Routh, temos:

\(s^3\) \(1\) \(2+K\)
\(s^2\) \(3\) \(6K\)
\(s^1=s\) \(- \displaystyle\frac{\begin{vmatrix} 1 & 2 + K \\ 3 & 6K \end{vmatrix} }{3} = \displaystyle\frac{6 - 3K}{3}\) \(0\)
\(s^0=1\) \(6K\)  

Como os elementos da primeira coluna devem ser positivos:

\[ \begin{align*} \displaystyle\frac{6 - 3K}{3} > 0\quad &\Rightarrow\quad K \lt 2,\\ 6K > 0\quad &\Rightarrow\quad K \gt 0. \end{align*} \]

Assim, para \(0 \lt K \lt 2\) o sistema é estável.
